# Forge v0.1.0

Forge turns a fuzzy coding request into shipped, reviewed code.

`prompt.md -> research.md -> plan.md -> build -> review.md`

## Highlights

- Added a five-stage workflow for Codex and Claude
- Added installers for macOS, Linux, and Windows PowerShell
- Added a git-aware revert helper for downstream stage commits
- Added example workflows for feature work, refactors, and bugfixes
- Added launch materials for promotion and community submissions

## Why It Exists

AI coding tools are strong at implementation but weak at forcing clear task definition, grounded research, and an explicit review loop. Forge makes those steps concrete and resumable.

## Included

- `SKILL.md`
- `install.sh`
- `install.ps1`
- `scripts/revert_stage_commits.py`
- `examples/`
- `docs/launch.md`

## Install

```bash
./install.sh both
```

## Try It

```text
$forge Add CSV and JSON export for invoices
$forge Continue from research.md
$forge Continue from research.md but stop after plan.md
```

## Good Fits

- Product features that need requirements clarity before coding
- Refactors where reviewability and rollback matter
- Bugfixes where repository-grounded research is more important than speed alone
